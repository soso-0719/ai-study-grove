# AI Study Grove

AI Study Grove is a study support application that combines focus timer, study logs, and AI-based review feedback.

## Concept

集中して学習し、記録し、AIで振り返るための学習支援アプリです。

## Planned Features

- Focus timer
- Study logs
- AI feedback
- Review questions
- Dashboard
- Learning history
