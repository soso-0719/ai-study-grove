# AI Study Grove

AI Study Grove は、学習記録を残すだけでなく、学んだ内容をAIで振り返りやすくする学習支援アプリです。

## 作ろうと思った理由

勉強をしていると、「今日は何をやったか」は残せても、「何を理解できて、何がまだ曖昧なのか」までは残りにくいと感じました。

特にプログラミングやIT基礎の学習では、分かったつもりでも後から説明できないことがあります。そこで、学習メモをAIに整理してもらい、要約、苦手ポイント、復習問題として残せるアプリを作りたいと考えました。

このアプリでは、単なる記録ではなく、次の学習につながる振り返りを作ることを目指します。

## MVP

最初のMVPでは、以下の機能に絞って開発します。

- 学習ログを作成する
- 学習ログ一覧を表示する
- 学習ログ詳細を表示する
- 学習ログを編集する
- 学習ログを削除する
- ダミーAIで要約、弱点、復習問題を生成する
- AIフィードバックをDBに保存する

最初から本物のAI APIは使わず、まずはダミーAIで画面、API、DBの流れを完成させます。アプリの形ができてから、OpenAI APIやGemini APIへの接続を検討します。

## 使用予定技術

| 分野 | 技術 |
|---|---|
| Frontend | Next.js, React, TypeScript |
| Backend | Python, Flask |
| DB | SQLite |
| AI | 最初はダミーAI、本実装でAI API連携 |
| Version Control | Git, GitHub |

## 想定する使い方

```text
1. 学習した内容をログとして登録する
2. 学習メモをもとにAIフィードバックを生成する
3. 要約、苦手ポイント、復習問題を確認する
4. 次回の学習時に復習材料として使う
```

## 開発方針

- 最初から大きく作りすぎない
- MVPを早く完成させる
- AI APIの料金を抑えるため、最初はダミーAIで作る
- READMEと設計メモを残し、考えたことが伝わるようにする
- 学習用ではなく、ポートフォリオとして見せられる完成度を意識する
